A hacking game inspired by Zelda, and Hack n' Slash.

This is only a beta, and will be discontinued once the first beta version is released.

Game

This game is simple - you hack enemies and/or fight them. Equipped with your handy CDCGR module and a Copper Broadsword, you set out to escape the world you find yourself in.
